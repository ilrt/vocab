Data Vocabulary Repository

Welcome to vocab.bris.ac.uk, the University of Bristol's data vocabulary repository (DVR).

This public repository has been established by IT Services Research and Development / ILRT to provide resolvable web addresses (URIs) for metadata vocabularies published by the University. The repository allows both software and developers to obtain further information about metadata used to describe data in University-developed applications and data sets.

For further details, please contact Simon Price or Mike Jones, IT Services R&D / ILRT.
